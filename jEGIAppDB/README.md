# jEGIAppDB

<h2>Requirements</h2>
jEGIAppDB uses:
- Jersey-client (v1.8)
- xml-apis (v2.0.2)

<h2>Install</h2>
Add the following dependencies to your pom.xml file:

    <dependency>
        <groupId>com.sun.jersey</groupId>
        <artifactId>jersey-client</artifactId>
        <version>1.8</version>
    </dependency>

    <dependency>
        <groupId>xml-apis</groupId>
        <artifactId>xml-apis</artifactId>
        <version>2.0.2</version>
    </dependency>
